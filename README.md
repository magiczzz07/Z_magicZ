## 🌟 Giới thiệu
**Z_magicZ** (trước đây là RiceGo) là một phần mềm Portable nhỏ gọn được thiết kế để quản lý và vận hành nhiều tài khoản Zalo cùng lúc (Zalo Clones) trên Windows 10/11. Bằng cách sử dụng cơ chế tạo các người dùng Windows (Windows Users) ẩn, Z_magicZ cách ly hoàn toàn bộ nhớ đệm và dữ liệu của từng tài khoản, đảm bảo chúng không bao giờ bị xung đột hay đăng xuất chéo.

## ✨ Tính năng nổi bật
* **Không cần cài đặt:** Chạy trực tiếp dưới dạng Portable (.exe).
* **Quản lý tập trung (Dashboard):** Giao diện Bento tối giản, thân thiện, dễ nhìn.
* **Cách ly dữ liệu 100%:** Mỗi clone chạy trên một phiên bản Windows User riêng.
* **Theo dõi trạng thái thời gian thực:** Hiển thị tự động tài khoản nào đang hoạt động.
* **Khởi động Zalo trong 1 click:** Tạo Shortcut tự động ra màn hình chính, hoặc mở từ danh sách.

## 🚀 Hướng dẫn sử dụng
### Bước 1: Chuẩn bị
Tải về file `.zip` từ mục Release hoặc thư mục gốc. Giải nén vào một ổ đĩa còn trống (Khuyên dùng ổ `D:\` hoặc `E:\` thay vì ổ `C:\` để tránh đầy bộ nhớ máy tính).

### Bước 2: Thiết lập đường dẫn (Chỉ làm 1 lần)
1. Mở file `Z_magicZ.exe`.
2. Bấm vào nút **Cài đặt** (Icon bánh răng) góc trên bên phải.
3. Ở mục **Thư mục Zalo gốc**, nhấn **Tự dò** để phần mềm tự tìm bản Zalo đang được cài trên máy tính (hoặc tự trỏ đến `C:\Users\<Tên_Bạn>\AppData\Local\Programs\Zalo`).
4. Ở mục **Thư mục lưu clone**, chọn nơi bạn muốn lưu các bản Zalo phụ (Vd: `D:\ZaloClone`).
5. Bấm **Lưu cài đặt**.

### Bước 3: Tạo Clone Zalo mới
1. Trên giao diện chính, bấm nút **+ Tạo Clone Zalo Mới**.
2. Nhập **Tên hiển thị** (Ví dụ: *Zalo Bán Hàng 1*).
3. Đặt **Tên Windows user** và **Mật khẩu** (Tùy chọn, để bảo mật dữ liệu).
4. Nhấn **Tạo Clone** và đợi khoảng 10-30 giây để hệ thống thiết lập không gian độc lập.

### Bước 4: Sử dụng
* Mỗi Clone mới tạo ra sẽ nằm trong danh sách ở Dashboard.
* Bấm **Mở Zalo** trên Clone tương ứng để đăng nhập và sử dụng.
* Quét mã QR bình thường như Zalo chính!
